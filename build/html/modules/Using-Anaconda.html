Using Anaconda in Server

Make sure we are in bash terminal

bash

To use anaconda

source /home/anaconda3/bin/activate

We recommend creating a separate conda environment for each project

conda create --name NAME

To enter the virtual environment

conda activate NAME

You will see that the terminal has (NAME) at the beginning of your input line

To install packages

conda install PACKAGE
